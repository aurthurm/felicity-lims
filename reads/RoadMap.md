# Roadmap

## Overview

The roadmap provides strategic direction for Felicity LIMS development. Due to evolving technologies and user expectations, future needs are continuously assessed.

## Recent Releases

### v0.2.3 (2026-02-14)

- UI/UX refresh across navigation and admin flows
- Tailwind v4 migration and design-system consistency
- Tenant/laboratory context validation and security middleware improvements
- Dashboard and analytics improvements (inventory KPI CSV export)
- Sample and billing workflow improvements (cost visibility, derived samples)
- Reflex editor reliability fixes
- Python 3.14 compatibility

### v0.2.2 and earlier

- Core LIMS functionality
- Multi-tenant architecture
- HIPAA-compliant encryption
- Microbiology module
- Instrument integration
